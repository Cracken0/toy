## linux

* apt

  > 换源：
  >
  > 中科大一键换源(搜 ustc ubuntu)
  >
  > `sudo sed -i 's@//.*archive.ubuntu.com@//mirrors.ustc.edu.cn@g' /etc/apt/sources.list`
  >
  > 换完更新
  >
  > ```sh
  >  sudo apt update # 更新列表
  >  #sudo apt upgrade # 更新软件 （谨慎使用 批量升级软件可能导致一些软件之间的版本依赖对不上）
  > ```
  >
  > 
  >
  > 另：
  >
  > 文件位置`/etc/apt/source.list`
  >
  > 本节均为 Ubuntu 20.04 的镜像源列表。若为其他版本，将所有`focal`更改为其他版本代号即可。
  >
  > 常用的Ubuntu版本代号如下：
  >
  > - Ubuntu 22.04：jammy
  > - Ubuntu 20.04：focal
  > - Ubuntu 18.04：bionic
  > - Ubuntu 16.04：xenial
  >
  > > Ubuntu 通常采用“形容词+小动物”作为版本代号（默认的壁纸），在镜像源列表中只有第一个词。
  >
  > 此外，默认注释了代码源以提高速度，注释了预发布软件源（可能不稳定）。如有需要可以取消注释。
  >
  > 建议将所有常用镜像源保存在`/etc/apt`目录下，并命名为类似`source.list.aliyun`的形式，需要使用时直接复制替换`source.list`文件即可。
  >
  > ```
  > #阿里云
  > deb http://mirrors.aliyun.com/ubuntu/ focal main restricted universe multiverse
  > deb http://mirrors.aliyun.com/ubuntu/ focal-security main restricted universe multiverse
  > deb http://mirrors.aliyun.com/ubuntu/ focal-updates main restricted universe multiverse
  > deb http://mirrors.aliyun.com/ubuntu/ focal-backports main restricted universe multiverse
  > 
  > # deb-src http://mirrors.aliyun.com/ubuntu/ focal main restricted universe multiverse
  > # deb-src http://mirrors.aliyun.com/ubuntu/ focal-security main restricted universe multiverse
  > # deb-src http://mirrors.aliyun.com/ubuntu/ focal-updates main restricted universe multiverse
  > # deb-src http://mirrors.aliyun.com/ubuntu/ focal-backports main restricted universe multiverse
  > 
  > ## Pre-released source, not recommended.
  > # deb http://mirrors.aliyun.com/ubuntu/ focal-proposed main restricted universe multiverse
  > # deb-src http://mirrors.aliyun.com/ubuntu/ focal-proposed main restricted universe multiverse
  > 
  > ```
  >
  > ```
  > #清华
  > deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ focal main restricted universe multiverse
  > deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ focal-updates main restricted universe multiverse
  > deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ focal-backports main restricted universe multiverse
  > deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ focal-security main restricted universe multiverse
  > 
  > # deb-src https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ focal main restricted universe multiverse
  > # deb-src https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ focal-updates main restricted universe multiverse
  > # deb-src https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ focal-backports main restricted universe multiverse
  > # deb-src https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ focal-security main restricted universe multiverse
  > 
  > ## Pre-released source, not recommended.
  > # deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ focal-proposed main restricted universe multiverse
  > # deb-src https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ focal-proposed main restricted universe multiverse
  > 
  > ```
  >
  > 安装软件包时自动装依赖
  >
  > `apt install -f`

## pyenv

```shell
curl https://pyenv.run | bash
echo 'export PATH="$HOME/.pyenv/bin:$PATH"' >> ~/.bashrc
echo 'eval "$(pyenv init --path)"' >> ~/.bashrc
echo 'eval "$(pyenv virtualenv-init -)"' >> ~/.bashrc
source ~/.bashrc
pyenv install 3.8.10
pyenv global 3.8.10
python --version

#换源 
export PYTHON_BUILD_MIRROR_URL="https://npm.taobao.org/mirrors/python/"
export PYTHON_BUILD_MIRROR_URL="https://registry.npmmirror.com/-/binary/python"
pyenv update

#临时解决
export v=3.8.10
wget https://registry.npmmirror.com/-/binary/python/$v/Python-$v.tar.xz -P ~/.pyenv/cache/
pyenv install $v 

#批量装python的依赖库
pip install -r <requirements.txt>
```

## git

```shell
git config --global user.name "RuixinHuang"
git config --global user.email "ruixin.huang@stepai.cn"
git fetch <-v|-all> #从远端更新分支情况到本地
git status #查看文件改动状态、分支
git checkout #切换到其他分支
git switch -c new-feature-branch # 创建一个新的本地分支
git checkout a -- x.c # 2. 从 A 分支获取 x.c 并覆盖当前分支的同名文件
git pull #拉取分支
git clone [-b <分支名>] <link of git> # 适用于public仓库
git clone git@github.com:Cracken0/toy.git # 适用于配置了ssh之后拉取private仓库
git rebase <分支名> #变基
git push -f # rebase完之后的提交要加-f才能成功 把-f换成--force-with-lease更安全
git diff <分支名> #验证差异
git diff origin/main --stat        # 仅显示变更的文件列表和行数统计
git add <file> #添加改动到暂存区
git commit -m "描述" #提交更改
git push #上传更改
git restore <file> #丢弃更改
git diff --staged  # 查看暂存区差异

ssh -T git@github.com # 测试连通性
# 如果出现Connection closed by 198.18.0.92 port 22
# 可能是因为代理
# 在 {用户}/.ssh/ 目录下建立config文本文档
Host github.com
Hostname ssh.github.com
Port 443
```

### perf

```sh
sudo apt install linux-tools-common
# 如果出现 You may need to install the following packages for this specific kernel:推荐安装可以按照提示将推荐安装包全部安装好
sudo apt-get install linux-tools-对应版本-generic linux-cloud-tools-对应版本-generic 

# or 从源码安装
uname -r # 查看内核版本
wget https://github.com/microsoft/WSL2-Linux-Kernel/archive/refs/tags/linux-msft-wsl-5.15.167.4.tar.gz #下载需要的版本
tar -zxvf linux-msft-wsl-5.15.167.4.tar.gz
cd WSL2-Linux-Kernel-linux-msft-wsl-5.15.167.4/tools/perf
make
# ----------------------------

sudo perf record -F 99 -a -g <bin> <args>
 -F 99 
表示采样的频率 
 -a
录取所有CPU的事件
 -g
使能函数调用图功能
sudo perf report # 生成报告预览
sudo perf report -n --stdio  # 生成消息报告
sudo perf script > out.perf # 导出文件

git clone --depth 1 https://github.com/brendangregg/FlameGraph.git
FlameGraph/stackcollapse-perf.pl out.perf > out.folded
FlameGraph/flamegraph.pl out.folded > out.svg

```
